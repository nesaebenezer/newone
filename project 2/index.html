Crime Clusters (7-day sliding window)

Detected Crime Clusters

Sliding Window Cluster Detection

Algorithm Steps

  1. Data Preparation: Group crimes by date
  2. Window Creation: Create 7-day sliding windows
  3. Frequency Calculation: Count crimes per window
  4. Threshold Detection: Identify above-average periods
  5. Cluster Classification: Rank by activity level

Technical Details

  • Time Complexity: O(n) where n = number of days
  • Space Complexity: O(n) for date storage
  • Window Size: Configurable (3, 7, 14 days)
  • Threshold: Above dataset average
  • Output: Ranked list of high-activity periods
